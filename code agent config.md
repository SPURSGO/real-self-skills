## Claude Code

- 官网参考：[Claude Code overview - Claude Code Docs](https://code.claude.com/docs/)
- 基础配置文件：~/.claude.json，位置在用户目录下，包含了很多CC的配置，如：
  - projects/根(用户级) 节点下的 mcpServers 节点，用于配置MCP服务
- skills 目录：~/.claude/skills/xxx-skill/
- 系统 rules：~/.claude/rules
